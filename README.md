# 🚀 Frontend React - MyApp

Frontend built with **React (Vite + TypeScript)** for interacting with MyApi backend.

---

## 🧱 Tech Stack

- React (Vite)
- TypeScript
- Axios
- Tailwind CSS
- React Router

---

## ✨ Features

### 🔐 Authentication
- Login with JWT
- Auto refresh token (silent)
- Auto logout if session invalid

### 📦 Product
- List products
- Create, update, delete product
- Pagination & search
